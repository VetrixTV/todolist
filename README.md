# Todolist

## How to use it

### Requirements

One of the following

- Docker
- Python

### Start the app

<!--- It just works -->
#### Docker
```bash
$ docker-compose up  -d
```

#### Python


Install the Library`s
```bash
$ pip install -r ./todolist/requirements.txt
```

Start the app
```bash
$ python ./todolist/main.py
```
